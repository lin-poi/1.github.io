java环境设置及eclipse安装

​ 嘛~作者花了好久终于搭好了环境，安好了eclipse，颇感不易啊。。。为了让小伙伴们不再和作者一样被各种文章绕来绕去，作者决定自己写一篇教程（PS：作者是win10 64位操作系统）。还有，请小伙伴们注意发表时间，作者被很多陈年老帖子搞得晕头转向，如果该文已过时，仅供参考。

​ 看了那么多教程想必大家见的最多的两组词：jdk和jre，java和javac。作者先引用两篇文章加以区分，当然如果你急于安装请跳过直接看正文。

java与javac

环境设置

​ 如果看过jdk与jre的区分，我们就知道只需要下载安装jdk就可以了

下载好后安装，记住安装路径

打开此电脑属性

打开环境变量，在系统变量中

1.添加JAVA_HOME:路径为你刚才安装jdk的路径

2.添加CLASSPATH路径为.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar （PS：一定记得路径前面有个.）

3.在PATH中添加 jdk路径\bin 并置顶（PS：win10这里不能用相对路径，必须用绝对路径）

最后记得点确定

右键主菜单

打开windows powershell（管理员）

输入java 显示如图则为安装成功

继续输入javac 显示如图则为安装成功

最后输入java -version（PS：java与-之间有空格）显示如图则为安装成功

至此，java运行环境搭建完成。

eclipse安装

​ 首先下载eclipse

注意选择32位或者64位，一定要与所下载的jdk保持一致。

安装后打开，第一次要设置存储路径并记得点对勾（意为记住我的选择）

点击左上角file，new，Java project并输入项目名称

然后继续file，new，class

1.最上方选择所创建项目下的src

2.填写程序名称

3.如果是主程序选择第一项

然后开始一起学习java吧！

常见问题汇总

这里仅就作者自己遇见过的问题作一些解答

1.环境变量配置完后在Windows powershell界面中java运行成功而javac运行失败

建议找环境变量中PATH部分的问题尤其不能使用相对路径

2.安装eclipse后打开总是链接到一个网址

eclipse下载错误（你下的那个估计是四十几M的）点击DOWNLOAD 64 bit下方的DOWNLOAD packages然后在里面下载eclipse ide for Java developers注意对应操作系统

3.JDK配置好后，打开eclipse后显示

这是JDK与eclipse操作位数不匹配导致的，在windows powershell中输入Java -version查看操作位数，若显示64bit则为64位，不显示则为32位，同时注意在此电脑的属性中查看自己的操作位数。
